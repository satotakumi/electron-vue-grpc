# electron-vue-grpc

> electron-vue で gRPCを使うサンプル


#### サーバ 起動方法
```bash
# ライブラリを取得
yarn # or npm install

# サーバを起動
yarn run server:start
```

#### クライアント 環境構築

``` bash
# ライブラリを取得
yarn # or npm install

# electron-rebuildする
./node_modules/.bin/electron-rebuild

# クライアントを起動
yarn run dev

```
